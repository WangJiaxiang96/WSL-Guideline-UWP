
# WSL Guideline 

### 这是WSL（Win10的Linux子系统）的进阶使用指南，在这里，你将学习到：  

1. 如何为WSL安装桌面环境（以运行图形界面程序）  

2. 解决WSL中常见问题

3. 用脚本部署一些自动化操作

&emsp;&emsp;......

### 在开始之前，你需要知道：

1. 使用WSL意味着你需要了解Linux系统的基础操作，而“WSL Guideline”**不会**讲解Linux系统的基础操作。

2. WSL虽然强大，但是作为子系统仍有一些底层的限制，WSL**不能**完全替代Linux发行版
